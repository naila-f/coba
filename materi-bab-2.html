Pengertian SQL

SQL ( Structured Query Language) atau juga disebut sebagai query adalah suatu bahasa yang digunakan untuk mengakses data dalam software DBMS. Bahasa ini merupakan bahasa standar yang memungkinkan untuk berkomunikasi dengan basis data untuk memperoleh informasi dari basis data. SQL dikenalkan pertama kali dalam IBM pada tahun 1970 dan sebuah standar ISO dan ANSII ditetapkan untuk SQL. Standar ini tidak tergantung pada mesin yang digunakan (IBM, Microsoft atau Oracle). Hampir semua software DBMS mendukung bahasa ini melakukan pengelolaan data, sehingga perintah SQL pada software hampir sama. Instruksi atau perintah SQL dapat dikelompokkan menjadi 3 yakni kelompok DDL, DML, DCL.

MySQL

MySQL adalah sebuah program database server yang dapat mengirimkan dan menerima data dengan cepat, multi user serta menggunakan perintah dasar SQL (Structured Query Language). MySQL pertama kali dirintis oleh seorang programmer database bernama Michael Widenius. MySQL merupakan database server yang free, artinya kita dapat menggunakan program database ini untuk keperluan pribadi atau usaha tanpa harus membeli atau membayar lisensinya. Database MySQL ini merupakan suatu program database yang berbentuk RDBMS (Relational Database Management System) yang menggunakan bahasa SQL

Menjalankan Aplikasi MySQL melalui CLI

Untuk dapat menggunakan program MySQL, terlebih dahulu mengaktifkan server MySQL dengan mengaktifkan MySQL. Contoh program apikasi MySQL yang digunakan adalah XAMPP 3.2, maka untuk menjalankan aplikasi MySQL yang terdapat pada direktori yaitu C:\Program Files\Xampp\MySQL\Bin
Berikut langkah-langkah untuk menjalankan melalui command prompt :

  1. Buka Xampp kemudian aktifkan server MySQL dengan klik tombol start
  2. Pastikan status dari MySQL adalah “running” yang berarti telah aktif
  3. Klik menu start atau klik pada gambar windows kemudian ketikkan cmd
  4. Jalankan perintah MySQL dengan command line sebagai berikut

    Penjelasan : mysql -u root -p , -u = user , root = nama user , -p = password apabila ada
